# ytcracker

downloading yt playlists in 10 lines of python.

- no shady website
- no shady download
- no ads
- can run inside containers
- watch offline /wo three letter agencies watching
- 10 lines (easy to audit)

## prerequesites

- python

## usage

fill any number of playlists to download inside `playlist.txt`.

```bash
python -m venv venv

source venv/bin/activate

pip install -r requirements.txt

python main.py
```
